# hydra

Personal rendering engine, with a v


## Installation


### Desktop Installation
```
git clone https://github.com/eqdiag/hydra
cargo run
```

### Web Installation
```
git clone https://github.com/eqdiag/hydra
wasm-pack build --target web
```
